Новая привычка

☁ Синхронизация между устройствами

Проверяю…
Разово, ~5 минут:
1. На supabase.com войди через GitHub → New project.
2. SQL Editor → вставь SQL (кнопка ниже) и запусти.
3. Authentication → Sign In / Providers → Email: выключи Confirm email (тогда вход без писем и лимитов).
4. Settings → API Keys: скопируй Project URL и ключ publishable (не secret!) сюда.
Первый раз — «Регистрация». На других устройствах тем же email и паролем — «Войти».
